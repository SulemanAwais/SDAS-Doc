## Requirement
SDAS can be installed in any android smartphone which have the following sensors available in it:
- **Accelerometer** 
 - **Gyroscope** 
 - **Magnetometer**
 - **Linear Acceleration**
 - **Gravity**
## Install with QR code
To install SDAS in your smartphone, scan the given QR code
